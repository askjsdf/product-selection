## 命名规范

```
images/
├── 01-KONG-Classic产品图.jpg
├── 02-OutwardHound产品图.jpg
├── 03-Chuckit产品图.jpg
├── 04-KONG-Wobbler-listing主图.jpg
├── 05-PetGenius-Ball-listing主图.jpg
├── ...
└── 28-APP界面截图.png
```

**注意事项**:
1. 所有文件名使用中文，方便识别
2. 编号01-28，便于代码中统一引用
3. 格式优先使用JPG（照片类），PNG（截图/透明背景类）
4. 所有图片需优化压缩，单张不超过500KB

---

## 使用方法

1. 按照清单准备28张图片
2. 放入 `/images/` 目录
3. 按照编号和文件名严格命名
4. 通知开发人员，将在代码中添加对应的 `<img>` 标签占位符
5. 后续只需替换图片文件，无需修改代码
