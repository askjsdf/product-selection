## 一、阶段定位与目标

### 1.1 业务目标

**核心问题**：在480个"品类×市场×平台"组合中,哪些值得深入分析?

**解决方案**：
- 📝 **行业天花板预判**：在筛选组合前，先理解行业规模、增长空间、消费人群特征，避免进入"天花板低"的品类
- 基于MCDA 5维度评分,快速筛选出Top 20-30个高潜力机会
- 通过可视化热力图,让用户直观看到"机会分布地图"
- 支持用户自主探索(点击查看详情)和系统推荐(Top 20清单)两种模式

### 1.2 输入与输出

| 项 | 内容 |
|-----|------|
| **输入** | 480组合数据库(20品类×6国家×4平台) + MCDA评分引擎 |
| **处理** | 5维度评分 → 综合排序 → 可视化展示 → 用户筛选 |
| **输出** | **机会候选池**(1-5个组合) |

### 1.3 关键成功指标(POC演示)

- ✅ 用户能在30秒内理解热力图的含义
- ✅ 用户能快速找到符合自己资源禀赋的机会
- ✅ 用户能理解"为什么这个组合得分高"的逻辑
- ✅ 候选池中的组合有明确的标签和特征

## 🆕 2.2 行业天花板预判分析

### 2.2.1 功能定位与底层逻辑

**核心问题**：在深入分析480个组合之前，我该不该进入这个行业？

**卖家的真实思考路径**：
```
宠物喂食器看起来不错，但：
- 整个美国宠物用品市场有多大？是成熟市场还是增长市场？
- 未来5年会增长吗？还是已经见顶了？
- 主力消费人群是谁？我的产品能不能匹配他们的需求？
- 这个行业的竞争格局如何？新品还有机会吗？
- 产品成熟度如何？需要多高的运营门槛？

如果行业天花板太低、增长停滞、或者运营门槛太高，
那就算某个具体组合评分高，也不值得进入。
```

**设计思路**：
- 不做"机械的行业报告"，而是提炼出**4个核心决策维度**
- 每个维度用**信号灯系统**（🟢黄灯🔴）快速判断
- 给出**进入建议**和**风险预警**

### 2.2.3 四维度信号灯判断逻辑

**维度1：行业规模与增长空间**

| 指标 | 🟢 绿灯 | 🟡 黄灯 | 🔴 红灯 |
|-----|--------|--------|--------|
| **5年CAGR** | >8% | 3-8% | <3% |
| **市场规模** | >$100M | $20-100M | <$20M |
| **趋势** | 稳定上升 | 波动上升 | 下降/停滞 |
| **结论** | 高增长潜力 | 稳定增长 | 成熟/衰退 |

**维度2：目标消费人群特征**

| 指标 | 🟢 绿灯 | 🟡 黄灯 | 🔴 红灯 |
|-----|--------|--------|--------|
| **年轻化比例** | >40%在18-40岁 | 20-40% | <20% |
| **消费能力** | 高支出意愿 | 中等 | 价格敏感 |
| **趋势特征** | 明确升级需求 | 稳定需求 | 需求萎缩 |
| **结论** | 人群画像清晰 | 需进一步细分 | 市场不明确 |

**维度3：市场结构与新品机会**

| 指标 | 🟢 绿灯 | 🟡 黄灯 | 🔴 红灯 |
|-----|--------|--------|--------|
| **垄断度** | <3 | 3-5 | >5 (一票否决) |
| **Top 10集中度** | <40% | 40-70% | >70% |
| **新品占比** | >20% | 10-20% | <10% |
| **结论** | 蓝海/新手友好 | 竞争适中 | 红海/垄断 |

**维度4：运营成熟度与门槛**

| 指标 | 🟢 绿灯 | 🟡 黄灯 | 🔴 红灯 |
|-----|--------|--------|--------|
| **A+覆盖率** | <70% | 70-90% | >90% |
| **视频覆盖率** | <60% | 60-85% | >85% |
| **认证要求** | 无特殊认证 | 常规认证(UL/CE) | 复杂认证(FDA药品类) |
| **专利风险** | 低 | 中等 | 高 |
| **结论** | 门槛低 | 需一定能力 | 高门槛 |

### 2.2.4 与后续模块的逻辑衔接

**天花板分析 → 热力图筛选的逻辑**：

```
用户操作流程:
1. 进入Step 1，首先看到"行业天花板分析"
2. 选择感兴趣的品类(如"宠物用品")
3. 查看4个维度的信号灯判断
4. 如果综合判断是🟢或🟡 → 点击"进入热力图选组合"
5. 如果综合判断是🔴 → 切换品类重新分析

系统逻辑:
- 天花板分析通过 → 热力图中该品类的组合高亮显示
- 天花板分析不通过 → 热力图中该品类的组合置灰/隐藏(可选项)
- 组合详情页会引用天花板分析的结论(如"该品类5年CAGR 8.61%")
```

**数据复用逻辑**：
- 天花板分析中的"行业规模"数据 → 映射到MA(市场吸引力)评分
- "垄断度"数据 → 映射到CF(竞争可行性)评分
- "认证要求""专利风险" → 映射到RC(风险控制)评分

**🆕 与Step 2/Step 5的层级差异说明**：

在6步工作流中,有3个层次的分析粒度,容易混淆,需要明确区分:

```
┌───────────────────────────────────────────────────────────┐
│ 三个层次的行业/市场/竞品分析                                 │
├───────────────────────────────────────────────────────────┤
│                                                            │
│ 🌍 Step 1 (宏观) - 行业天花板分析                           │
│  ├─ 分析对象: 整个行业(如"宠物用品行业")                     │
│  ├─ 核心问题: 这个行业值不值得进入?                          │
│  ├─ 数据粒度: 行业级聚合数据                                │
│  ├─ 关键指标: 行业规模、5年CAGR、目标人群特征、运营门槛       │
│  └─ 输出: 🟢可进入 / 🟡谨慎进入 / 🔴不建议进入             │
│                                                            │
│ 📊 Step 2 (中观) - 品类洞察分析                             │
│  ├─ 分析对象: 品类内部结构(如"宠物喂食器"的细分)             │
│  ├─ 核心问题: 品类内部有哪些主流款式?增长机会在哪?           │
│  ├─ 数据粒度: 品类聚合数据(如"智能式占18%, +180%增长")      │
│  ├─ 关键指标: 产品类型占比、价格带分布、功能特征分布          │
│  └─ 输出: 推荐方向("智能式+$45-75+摄像头")                  │
│                                                            │
│ 🔍 Step 5 (微观) - 竞品深度拆解                             │
│  ├─ 分析对象: 具体竞品ASIN(如"Petlibro B09ABC123")         │
│  ├─ 核心问题: 具体竞品怎么做的?用户哪里不满?我如何做更好?     │
│  ├─ 数据粒度: ASIN级数据(评论、Q&A、图片、视频)             │
│  ├─ 关键指标: 差评分类、Q&A高频问题、图片风格、视频脚本       │
│  └─ 输出: 差异化策略("解决发热问题+强化旅行场景")            │
│                                                            │
└───────────────────────────────────────────────────────────┘

从"值不值得做" → "哪个细分机会大" → "怎么做得不一样"
层层递进,粒度越来越细
```

**为什么需要三个层次?**

1. **Step 1过滤掉天花板过低的行业**
   - 例如: CD播放器行业,5年CAGR -15%,市场持续萎缩 → 🔴不进入
   - 避免在"夕阳行业"浪费时间

2. **Step 2在可进入行业中找到结构性机会**
   - 例如: 宠物用品整体增长8.6% → Step 1通过
   - 但"重力式喂食器"价格战激烈,而"智能式"增长+180% → Step 2推荐智能式
   - 避免进入"红海细分",找到"蓝海切入点"

3. **Step 5在选定方向后找到差异化打法**
   - 例如: Step 2推荐"智能式",Step 3筛选出具体产品
   - 但竞品都在强调"APP控制",用户差评集中在"发热" → Step 5找到差异化点
   - 避免"同质化竞争",找到"差异化优势"

**三个层次的数据来源差异**：

| 层次 | 数据来源 | 数据示例 |
|-----|---------|---------|
| Step 1 (行业) | 行业报告、市场研究机构 | "美国宠物用品市场$206.7B (2027)" |
| Step 2 (品类) | 品类聚合数据、第三方工具 | "智能式喂食器占18%, +180% YoY" |
| Step 5 (竞品) | 平台爬虫、评论NLP分析 | "Petlibro产品评论中12%提到发热问题" |

**演示时如何体现层次区分**：

```
Demo脚本示例:

"Step 1告诉我们,宠物用品行业5年CAGR 8.6%, 增长潜力大,可以进入。

但进入后做什么?Step 2深入分析宠物喂食器这个品类,发现:
- 重力式占20%,价格战激烈,不推荐
- 智能式占18%,增长+180%,这就是机会!

那智能式具体怎么做?Step 5拆解Top竞品,发现:
- 12%用户抱怨发热,但竞品中只有10%明确宣传散热
- 这就是我们的差异化切入点!"

层层递进,从宏观到微观,逻辑清晰。
```

## 三、业务分析要点（核心内容）

### 3.1 卖家在机会发现阶段会关注哪些因素？

基于对跨境电商选品的理解,卖家在这个阶段通常会考虑：

#### 📝 3.1.0 行业天花板维度（前置判断）

**卖家的底层逻辑**：在投入精力分析具体组合之前，先判断这个行业值不值得进入。

- **行业规模与成长性**：
  - 当前市场规模多大？未来5年增长空间如何？
  - 行业是处于导入期、成长期还是成熟期/衰退期？
  - 细分类目在大盘中的占比？是否有独立增长空间？

- **消费人群特征**：
  - 主力消费人群是谁？年龄结构、消费能力、消费习惯？
  - 人群是否年轻化？（年轻人群意味着智能化、个性化需求）
  - 养宠/健身/美妆等人群渗透率？增长趋势如何？

- **市场结构与进入门槛**：
  - 品牌集中度如何？头部垄断还是分散竞争？
  - 新品在市场中的占比？新卖家能不能活下来？
  - 运营成熟度如何？A+/视频覆盖率高不高？
  - 需要什么认证？有无专利风险？

**为什么重要**：
- ❌ 错误：只看单个组合得分高就进入 → 可能进入"天花板低"或"高度垄断"的品类
- ✅ 正确：先判断行业天花板 → 再从合格品类中筛选最优组合

**真实案例**：
```
宠物喂食器单组合评分79分（不错），但：
- 行业大盘：美国宠物市场8.61% CAGR，天花板高 ✅
- 消费人群：49% < 40岁，年轻化明显，智能化需求强 ✅
- 市场结构：垄断度4（<5），新品占比18%，可进入 ✅
- 运营门槛：A+覆盖98%，需要内容能力，但专利风险高 ⚠️

综合判断：✅ 值得进入，但需准备好内容能力和专利申请
```

#### 3.1.1 市场维度
- **市场规模**：GMV够不够大？能容纳多少新卖家？
- **增长趋势**：是成熟市场还是增长市场？有没有红利期？
- **搜索热度**：用户需求是否旺盛？季节性如何？
- **市场成熟度**：是蓝海、红海还是血海？
- 📝 **市场容量量化**：
  - **月总销量**：该品类在该市场的月总销售件数（理想范围：能支撑新卖家分羹，又不至于过度饱和）
  - **供需比**：搜索量/商品数量（供需比越高，说明需求旺盛但供给不足，新手机会大）
  - **新品友好度**：近3个月新品占比和新卖家存活率（>20%说明市场对新进入者友好）

#### 3.1.2 竞争维度
- **集中度**：头部卖家占据多少份额？新手有没有机会？
- **竞品数量**：同类卖家有多少？竞争激烈程度？
- **新手存活率**：新卖家能活下来的比例？
- **价格战程度**：是否陷入价格战？利润空间如何？
- 📝 **垄断度识别**：
  - **垄断度**：头部月均销量/Top100月均销量（>5即高度垄断，新手难以突破，建议放弃）
  - **竞争程度**：销量/产品数量（越高说明每个产品分得的销量越多，竞争相对温和）
  - **一票否决机制**：垄断度>5的品类直接标红预警，避免新手陷入"看似热销实则垄断"的陷阱

#### 3.1.3 利润维度
- **毛利率**：扣除COGS后的毛利空间？
- **客单价**：单笔订单价值？影响广告ROI
- **转化率**：流量到订单的转化效率？
- **复购率**：是一次性消费还是重复购买？

#### 3.1.4 资源匹配维度
- **供应链要求**：MOQ多少？交期要求？自己能否满足？
- **内容能力**：TikTok等平台需要视频内容,有无能力？
- **资金要求**：首批备货+广告需要多少资金？
- **认证门槛**：需要什么资质认证？成本和周期？

#### 3.1.5 风险维度
- **合规风险**：类目是否有政策限制？
- **知识产权**：是否容易侵权？
- **退货率**：产品退货率高不高？
- **平台政策**：平台是否稳定？有无突然封店风险？

### 3.2 如何从480个组合中快速筛选？

**传统方法的痛点**：
- 逐个查看480个组合？效率太低
- 只看GMV？可能是红海市场
- 只看增长率？可能资源匹配不了
- 凭经验判断？容易错过机会或踩坑

**MCDA方法的优势**：
- **多维综合**：同时考虑市场、竞争、利润、资源、风险5个维度
- **量化评分**：每个维度0-100分,可比较
- **权重调整**：根据自身战略目标调整权重
- **一票否决**：避免"某方面很好但致命短板"的陷阱

**筛选逻辑示例**：

```
假设卖家画像:
- OEM工厂转型,有供应链优势
- 资金有限($5万预算)
- 希望快速验证,3个月见效
- 没有内容团队

权重设定:
- PP(利润)权重↑ 35% (资金有限,需要高ROI)
- RF(资源)权重↑ 25% (供应链优势要发挥)
- MA(市场)权重↓ 15% (不追求最大市场)
- CF(竞争)权重 20% (避开红海)
- RC(风险)权重 5% (可承受一定风险)

筛选结果:
会优先推荐"高利润+供应链友好+竞争温和"的组合,
而不是"市场巨大但竞争激烈或内容驱动"的组合
```

### 3.3 如何理解"机会"的本质？

**机会 ≠ 热销品**
- 热销品可能已是红海,后来者难以分羹
- 真正的机会是"需求未被充分满足"的市场

**机会 = 需求 × 供给缺口 × 资源匹配**
- **需求**：市场规模和增长(MA维度)
- **供给缺口**：竞争温和、存在差异化空间(CF维度)
- **资源匹配**：卖家有能力做好(RF维度)

**MCDA的5维度本质**：
- **MA**: 市场有没有？
- **CF**: 竞争激不激烈？
- **PP**: 能不能赚钱？
- **RF**: 我能不能做？
- **RC**: 风险可不可控？

5个问题都是"Yes"才是真正的机会。

## 六、关键设计原则

### 6.1 信息分层原则

**三层信息架构**：
1. **总览层**：热力图/Top 20,快速扫描
2. **详情层**：单组合的详细数据,深入了解
3. **对比层**：多组合横向对比,辅助决策

**用户可根据需要在三层间切换**,不强制线性浏览。

### 6.2 方法论可见原则

**让用户理解"为什么"**：
- MCDA的5个维度明确展示
- 每个维度的评分逻辑可查询
- AI洞察必须有数据支撑,不能黑盒

**示例**：
- ❌ "这是个好机会"（黑盒）
- ✅ "该组合MA得分85(GMV $58M,增长185%),CF得分76(CR5仅35%),综合评分82,属于高增长+低竞争的优质机会"（透明）

### 6.3 渐进式决策原则

**不强迫用户一次选定**：
- 候选池支持添加、移除、对比
- 用户可以在Step 1和Step 2之间往返
- 允许"先加入候选池,后面再决定"

### 6.4 业务导向原则

**所有设计服务于业务决策**：
- 数据展示不是目的,辅助决策才是
- 每个图表要能回答一个业务问题
- AI洞察要给出"建议做/不做"的明确结论
