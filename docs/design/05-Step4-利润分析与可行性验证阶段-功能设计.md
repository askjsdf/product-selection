## 一、阶段定位与目标

### 1.1 业务目标

**核心问题**：这个产品能赚钱吗？我能做吗？

**解决方案**：
- 基于GECOM全景成本模型，进行详细的成本与利润测算
- 识别关键风险点（认证、合规、供应链、季节性等）
- 输出可行性结论和建议

### 1.2 输入与输出

| 项 | 内容 |
|-----|------|
| **输入** | Step 3的产品候选池（3-10个产品） |
| **处理** | 成本核算 → 利润建模 → 风险评估 → 可行性判断 |
| **输出** | **精选产品清单**（1-3个通过验证的产品） |

## 二、核心功能模块（已实现）

### 2.1 GECOM全景成本模型

**成本维度**：
```
总成本 = COGS (采购成本)
        + 头程物流 📝 (需对比海运/空运/快递方案)
        + 平台费用 (佣金、FBA仓储费)
        + 关税与合规成本
        + 广告成本
        + 退货/换货成本
```

**🆕 卖家底层逻辑**：物流方案选择直接影响15-20%的利润率差异。

**真实案例对比**（宠物喂食器 - 单碗款）：
```
方案1：空运 (快速起量，但利润低)
├─ COGS: $8.50
├─ 空运物流: $2.80 (7-10天到仓)
├─ FBA+佣金: $5.21
├─ 广告: $1.35
└─ 售价$19.99 → 毛利$2.13 (13.62%) ⚠️ 利润薄

方案2：海运 (利润高，但周转慢)
├─ COGS: $8.50
├─ 海运物流: $1.15 (35-45天到仓)
├─ FBA+佣金: $5.21
├─ 广告: $1.35
└─ 售价$19.99 → 毛利$4.78 (26.17%) ✅ 推荐

差异分析：
→ 物流成本差: $2.80 - $1.15 = $1.65 (占售价8.25%)
→ 毛利率差: 26.17% - 13.62% = 12.55% (接近翻倍!)
→ 选择策略: 新品期可空运测试，稳定后切海运
```

**🆕 产业带信息**（供应链定位）

**底层逻辑**：不同产业带的供应链成熟度、MOQ、工期、认证能力差异巨大。

**示例：宠物喂食器产业带分布**：
```
产业带地图：
├─ 🏭 河北 (主产区)
│   特点: 塑料制品产业链完整，MOQ低(100-300件)
│   优势: 成本低，交期短(15-20天)
│   劣势: 设计创新能力弱，认证资源少
│
├─ 🏭 浙江 (宁波/温州)
│   特点: 五金+电子结合，智能化方案成熟
│   优势: FDA/UL认证渠道完善，品质稳定
│   劣势: MOQ较高(300-500件)，单价高10-15%
│
├─ 🏭 广东 (深圳/东莞)
│   特点: 智能硬件产业链，APP开发/芯片配套完善
│   优势: 智能款首选，迭代快，技术支持强
│   劣势: 价格最高，MOQ通常500+
│
├─ 🏭 山东
│   特点: 传统制造，适合简单款
│   优势: 价格实惠，MOQ灵活
│   劣势: 智能化能力弱
│
└─ 🏭 上海
    特点: 外贸型工厂，质量管控严格
    优势: 欧美市场经验丰富，认证齐全
    劣势: 价格偏高，订单周期长

产业带选择策略：
→ 低端款($10-20): 河北/山东 (性价比)
→ 中端款($20-40): 浙江 (品质+认证)
→ 智能款($40+): 深圳 (技术支持)
```

### 2.2 利润场景分析

**多场景建模**：
- 乐观场景（高销量、低广告成本）
- 基准场景（行业平均水平）
- 悲观场景（低销量、高广告成本）

**输出指标**：
- 单件利润
- 毛利率 / 净利率
- 盈亏平衡销量
- ROI预估

### 2.3 风险评估矩阵

**风险类型**：
- 🔴 高风险：合规/认证/专利
- 🟡 中风险：季节性/供应链/竞争
- 🟢 低风险：市场稳定/供应链成熟

### 2.4 可行性判断

**判断逻辑**：
- ✅ 可行：利润率>15%，风险可控，资源匹配
- ⚠️ 有条件可行：需要解决特定问题（如认证）
- ❌ 不建议：利润太薄或风险过高
