# Module 2.2: 数字化市场地图设计

## 案例品类：宠物用品 - 互动玩具细分市场
**承接上一步**: 从"五看洞察"锁定高潜力细分市场 → 互动玩具（Interactive Pet Toys）

---

## 设计目标

基于华为IPD-MM的市场地图理论,在电商语境下重新定义为"**流量-转化-交付**"的三维网络,揭示价值流动的动态路径。

**三层映射关系**:
1. 决策流映射（Buyer vs. User）
2. 渠道影响力映射（Channel Influence）
3. 竞争生态映射（Competitor Ecosystem）

## 技术实现备注

- 决策流程图: Mermaid.js / D3.js Sankey Diagram
- 渠道对比表: 自定义CSS Grid + 图标库
- 竞争生态图: D3.js Force-directed Graph
  - 节点大小: 根据市场份额动态计算
  - 连线粗细: 根据用户转换概率
  - 颜色编码: 按竞争威胁等级
- SWOT矩阵: 四象限布局 (Flexbox)

- 网络图节点点击: 弹出Modal显示竞品详细数据
- 渠道卡片悬停: 显示详细ROI计算公式
- 搜索关键词热力图: 按搜索量着色
- 预算分配滑块: 实时计算不同渠道组合的预期GMV

### 数据模拟
- 决策流数据: 基于真实用户行为研究论文
- 渠道ROI: 参考行业benchmark (Jungle Scout报告)
- 竞品ASIN: 使用真实Amazon数据
