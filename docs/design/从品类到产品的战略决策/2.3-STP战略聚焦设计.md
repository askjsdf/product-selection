# Module 2.3: STP战略聚焦设计

## 案例品类：宠物用品 - 互动玩具
**承接上一步**: 从市场地图锁定战略定位 → "现代化KONG替代品" → 执行STP精准切分

---

## 设计目标

基于STP理论（Segmentation-Targeting-Positioning），将宠物互动玩具市场科学切分为可执行的细分市场，锁定最具价值的目标人群，并明确品类角色定义。

**核心流程**:
1. **S**egmentation（市场细分）- 多维度切分市场
2. **T**argeting（目标选择）- 评估并选择目标细分市场
3. **P**ositioning（市场定位）- 确立差异化定位
4. **+** Category Role（品类角色定义）- 明确产品在业务版图中的使命
